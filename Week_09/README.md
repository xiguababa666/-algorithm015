# week09
- 不同路径 II
    - 动态规划思路
        - 起点为(0,0)，终点为(m,n)
        - 从后往前递推(i,j=m,n => i,j=0,0)，结果为dp[0][0]
        - obstacleGrid[i][j] == 1时(障碍位)，dp[i][j] = 0
        - obstacleGrid[i][j] == 0时(无障碍)，dp[i][j] = dp[i + 1][j] + dp[i][j + 1]  【超出边界部分按0算】
        - 可优化为一维
            - dp[n] = 1
            - if(obstacleGrid[i][j]==1) dp[j] = 0
            - if(obstacleGrid[i][j]==0) dp[j] = dp[j] + dp[j + 1]
